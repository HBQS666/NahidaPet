# 纳西妲桌宠
✨ 基于 Qt C++ 开发的轻量级桌面宠物

## 📝 项目介绍
本项目是一款轻量化桌面宠物程序，以原神角色纳西妲为原型，基于 Qt 框架实现无边框透明窗口与序列帧动画展示，支持自由拖拽、动作切换、系统托盘常驻，简洁美观、低占用、无打扰。

## 🎯 实现功能
- ✅ 序列帧动画流畅播放
- ✅ 鼠标左键自由拖拽移动
- ✅ 右键菜单切换动画动作
- ✅ 右键菜单支持隐藏窗口
- ✅ 系统托盘常驻（显示/退出）
- ✅ 无边框透明悬浮效果

## 🛠 开发环境
- **开发框架**：Qt 5 / Qt 6
- **开发语言**：C++
- **开发工具**：Qt Creator

## 🚀 使用方法
1. 使用 Qt Creator 打开项目并编译运行
2. 左键按住桌宠可拖动到任意位置
3. 右键打开菜单，切换 swing / act_1 动画
4. 选择「隐藏」可将桌宠收到系统托盘
5. 托盘右键可重新显示或退出程序

## 📁 核心文件
- `main.cpp` 程序入口、系统托盘
- `widget.h` 桌宠类、动作枚举、拖拽逻辑
- `widget.cpp` 动画绘制、菜单、资源加载
